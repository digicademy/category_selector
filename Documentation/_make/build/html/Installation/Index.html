Installation¶

Requirements¶

  • Made for TYPO3 6.1+
  • An extbase/fluid based list of objects

Installation¶

Import the extension from TER and install. Include the static TypoScript of this extension in your template. Use the category selector plugin to configure your category tree. Insert the Fluid widget into the list template that you wish to filter (see below).
